# Maquina-de-Refrigerante

Projeto de uma máquina de refrigerante feito com HTML, CSS e JavaScript.

## 👩‍💻 Autor
Bruna Izabel da Silva Pereira

## 🚀 Funcionalidades
- Escolher bebida
- Interface interativa
- Simulação de compra

## 📁 Arquivos
- `machine.html`
- `machine_style.css`
- `machine_refri.js`

## 💻 Como rodar
- Abra o arquivo `machine.html` no navegador.
- Clique no botão "Escolha Aqui".
- Escolha sua bebida e clique no botão "voltar".
- Arraste as moedas em cima do botão circular cinza indicado pela seta 
  até completar o valor.
